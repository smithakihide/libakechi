this is a library set written in C++ 11 and compatible with windows and linux.

you can use Visual Studio 2017 (or later), g++ and clang++/HCC to compile this library using makefile or VS project file.

To compile and use in Linux, please use makefile in the directory liba.

Vector.h includes linear algebra library.
strlib_akechi.h includes json parser, time recorder, UUID, and so on.

Note that you need the library of uuid-dev to link this library to you program.
